## Project Overview
EduLearn is an EdTech platform offering courses in Data Science, Cloud, and Web Development. This project builds a complete data engineering pipeline using AWS and Databricks.

## Architecture
GitHub → AWS S3 → Glue → Athena → Databricks (Bronze, Silver, Gold)

## Tech Stack
- AWS (S3, Glue, Athena, Redshift, Kinesis)
- Databricks (Delta Lake)
- Python
- SQL
